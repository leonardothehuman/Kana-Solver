copySrc -> Static assets that will be copied to package.nw directory
dist -> The final directory for built software and nw.js binaries
    dist/package.nw -> Main directory where nw.js finds the program
genereatedCopySrc -> same as copySrc, but everything there was generated by the regenerate script
generatorData -> the source directory with the source data for the regenerate script
node_modules -> where npm install dependencies
nwBinaries -> cache for the downloaded nw.js binaries
src -> source code
    src/generated -> the destination directory for auto-generated code
